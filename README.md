# Lang JS - Lang言語のJS実装

例の実行

```sh
deno task run examples/program.lng
```

## コメント

コメントは `;` で開始します

```
; this is a comment
```

## 演算

### 算術演算

二項演算子 `+`, `-`, `*`, `/`

単項演算子 `-`

- 単項演算子 + は存在しない
- 先頭にない単項演算子 - は括弧で囲む必要がある

```
+42      ; ❌ NG
2 * -3   ; ❌ NG
2 * (-3) ; ✅ OK
-2 * 3   ; ✅ OK
```

### 比較演算

`=`, `~=`, `<`, `<=`, `>`, `>=`
